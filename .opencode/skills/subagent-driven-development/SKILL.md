# Subagent-Driven Development

Delegates implementation tasks to sub-agents effectively.

## Process
1. **Decompose**: Break feature into sub-tasks
2. **Spec**: Write clear instructions per sub-task
3. **Delegate**: Assign to sub-agent with context
4. **Monitor**: Track progress and quality
5. **Integrate**: Merge sub-agent outputs
6. **Verify**: End-to-end test the integration

## Writing Good Instructions
- Specify the exact files to modify
- Provide context (existing patterns, conventions)
- Include acceptance criteria
- Set boundaries (what NOT to do)
- Reference relevant skills to load
