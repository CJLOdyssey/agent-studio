# Executing Plans

Follows structured plans with disciplined execution and verification.

## Process
1. Read the complete plan first
2. Execute steps in order
3. Verify each step before proceeding
4. Track progress and update plan status
5. Report deviations or blockers
6. Verify final result matches plan specification

## Rules
- Don't skip ahead
- If a step fails, stop and report
- Update plan with actual time/task estimates
- Mark completion status per step
