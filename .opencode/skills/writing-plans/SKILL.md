# Writing Plans

Creates structured implementation plans for complex work.

## Plan Structure
1. **Goal**: What we're building and why
2. **Requirements**: Functional and non-functional
3. **Approach**: High-level implementation strategy
4. **Steps**: Ordered list of implementation tasks
5. **Risks**: Known challenges and mitigations
6. **Acceptance Criteria**: How we know it's done

## Guidelines
- Break into small, verifiable steps
- Each step should produce a working increment
- Identify dependencies between steps
- Estimate effort per step
- Define clear completion criteria for each step
- Call out areas needing research or experimentation
