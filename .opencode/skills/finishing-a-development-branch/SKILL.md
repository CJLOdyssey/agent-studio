# Finishing a Development Branch

Completes feature branches properly before merging.

## Checklist
- [ ] All tests pass
- [ ] No lint errors
- [ ] Self-review completed
- [ ] Edge cases handled
- [ ] Error paths tested
- [ ] Console.log/debug code removed
- [ ] Comments and documentation updated
- [ ] CHANGELOG or release notes updated
- [ ] Branch rebased on target
- [ ] PR description written

## Process
1. Run full test suite
2. Run linter and type checker
3. Review all changed files
4. Clean up temporary code and debug statements
5. Write or update relevant docs
6. Create PR with clear description
