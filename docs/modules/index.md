# 模块文档索引

> DevAgents OS 全部业务模块拆解，每个模块包含：业务闭环、层级实现、数据流。

---

## 模块总览

| # | 模块 | 子模块 | 核心闭环 |
|---|---|---|---|
| 01 | [对话与会话管理](01-对话与会话管理/) | — | 需求输入 → Agent执行 → 流式输出 → 消息保存 → 历史查看 → 继续对话 |
| 02 | [团队管理](02-团队管理/) | — | 创建团队 → 添加成员 → 配置Agent → 保存 → 选择团队 → 发起对话 |
| 03 | [Agent执行引擎](03-Agent执行引擎/) | orchestration · streaming | 需求 → Prompt构建 → LangGraph轮转 → 收敛检测 → 结果保存 → Memory提取 |
| 04 | [记忆与知识管理](04-记忆与知识管理/) | extraction · rag | 对话结束 → 提取Memory → 保存 → 下次注入 → RAG增强 → 效果提升 |
| 05 | [API密钥管理](05-API密钥管理/) | — | 添加密钥 → 加密存储 → 测试连接 → 选择模型 → 调用LLM → 使用统计 |
| 06 | [工具与技能生成](06-工具与技能生成/) | tool-generation · skill-generation | 描述需求 → LLM生成 → 验证 → 保存 → Agent可调用 |
| 07 | [文件附件管理](07-文件附件管理/) | — | 选择文件 → 上传验证 → 存储 → 关联Run → Agent读取 |
| 08 | [命令面板](08-命令面板/) | — | 快捷键 → 命令列表 → 执行 → 日志记录 → UI响应 |
| 09 | [模型管理](09-模型管理/) | — | 配置密钥 → 提取模型 → 下拉框展示 → 用户选择 → 绑定Run |
| 10 | [系统团队管理](10-系统团队管理/) | tools-agent · skill-agent | 加载配置 → 列出Agent/工具/技能 → 生成 → 保存 → 主Agent调用 |
