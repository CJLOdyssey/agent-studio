# Part A 项目监控数据表

## 1. 引言

### 1.1 目的

本文档用于系统化采集项目执行过程中的关键监控数据，通过挣值管理（EVM）方法量化项目进度与成本绩效，并为偏差识别与控制提供数据基础。

本文档的编制目的包括：

- 建立统一的项目监控数据采集规范；
- 提供可复用的监控数据记录模板；
- 支撑挣值分析所需的 Planned Value（PV）、Earned Value（EV）、Actual Cost（AC）数据；
- 为偏差控制报告提供输入数据。

### 1.2 适用范围

本文档适用于[占位符]项目的全生命周期监控，覆盖以下阶段：

- 需求分析与定义
- 系统设计
- 编码实现
- 测试验证
- 部署上线
- 运维支持

### 1.3 术语与定义

| 术语 | 英文全称 | 定义 |
|------|----------|------|
| PV | Planned Value | 计划价值，截至当前日期按计划应完成工作的预算成本 |
| EV | Earned Value | 挣值，截至当前日期实际完成工作的预算成本 |
| AC | Actual Cost | 实际成本，截至当前日期实际发生的成本 |
| SPI | Schedule Performance Index | 进度绩效指数，SPI = EV / PV |
| CPI | Cost Performance Index | 成本绩效指数，CPI = EV / AC |
| SV | Schedule Variance | 进度偏差，SV = EV - PV |
| CV | Cost Variance | 成本偏差，CV = EV - AC |
| BAC | Budget at Completion | 完工预算，项目总预算 |
| EAC | Estimate at Completion | 完工估算，EAC = BAC / CPI |
| ETC | Estimate to Complete | 完工尚需估算，ETC = EAC - AC |
| VAC | Variance at Completion | 完工偏差，VAC = BAC - EAC |

## 4. 挣值分析

### 4.1 挣值管理（EVM）核心指标计算表

| 指标 | 计算公式 | 计算结果 | 说明 |
|------|----------|----------|------|
| PV（计划价值） | 按计划截至当前应完成工作的预算成本 | [占位符] | [占位符] |
| EV（挣值） | 实际完成工作的预算成本 | [占位符] | [占位符] |
| AC（实际成本） | 实际发生的成本 | [占位符] | [占位符] |
| BAC（完工预算） | 项目总预算 | [占位符] | [占位符] |
| SV（进度偏差） | EV - PV | [占位符] | SV > 0 进度超前，SV < 0 进度滞后 |
| CV（成本偏差） | EV - AC | [占位符] | CV > 0 成本节约，CV < 0 成本超支 |
| SPI（进度绩效指数） | EV / PV | [占位符] | SPI > 1 进度超前，SPI < 1 进度滞后 |
| CPI（成本绩效指数） | EV / AC | [占位符] | CPI > 1 成本节约，CPI < 1 成本超支 |
| EAC（完工估算） | BAC / CPI | [占位符] | 基于当前绩效的完工总成本预测 |
| ETC（完工尚需估算） | EAC - AC | [占位符] | 完成剩余工作还需的成本 |
