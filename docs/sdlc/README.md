## 流程顺序

```
01-立项 → 02-需求 → 03-设计 → 04-技术评审 → 05-编码 → 06-测试
   → 07-部署 → 08-验收 → 09-交付售后 → 10-运维 → 11-结项

贯穿全流程的过程管理：
   12-配置与构建管理  13-质量保证  14-项目监控  15-风险管理  16-培训管理
```

## 使用方式

每份文档均为 **通用模板**，使用 `[占位符]` 标记待填写内容。团队在具体项目中应将占位符替换为实际内容，并增删章节。
